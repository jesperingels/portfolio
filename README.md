# Portfolio (Next.js)

Minimale basis voor een persoonlijke portfolio-site in Next.js (App Router).

## Starten

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Routes in deze basis

- `/`
- `/about`
- `/work`
- `/articles`
- `/lab`
- `/contact`

Deze setup is bewust klein gehouden zodat je daarna gericht content, SEO en componenten kunt uitbouwen.
